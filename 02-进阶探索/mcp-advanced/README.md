## 模块概述

## 常见问题

### Q: 必须使用 TypeScript 吗？
**A**: 推荐使用 TypeScript，因为官方 SDK 是 TypeScript 编写的。但也可以使用其他语言，需要自己实现协议。

### Q: 开发的服务器可以商业化吗？
**A**: 可以，MCP 是开放协议。但需要遵守相关开源协议和法律法规。

### Q: 如何调试 MCP 服务器？
**A**: 可以使用 MCP Inspector 工具，或者在 Claude Code 中直接测试。
