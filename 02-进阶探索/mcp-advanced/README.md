## 模块概述

本模块深入学习 MCP 的高级特性，包括 Resources（资源）、Prompts（提示模板）等高级功能的配置和使用。**本模块聚焦于理解高级特性和配置现有服务器**，而非深入开发自定义服务器。

> **注意**：本模块已调整为「了解」级别（P3）。如果你对开发自定义 MCP 服务器感兴趣，可以选学完整模式。

## 常见问题

### Q: Resources 和工具有什么区别？
**A**: 工具是 Claude 可以调用的函数，Resources 是 Claude 可以访问的数据源。工具用于执行操作，Resources 用于获取信息。

### Q: 必须了解开发流程吗？
**A**: 不必须。本模块重点是配置和使用。开发流程只是理论了解，帮助你更好地理解服务器工作原理。

### Q: 如何配置多服务器协作？
**A**: 在 Claude Code 的配置文件中添加多个 MCP 服务器配置，每个服务器可以有独立的工具和资源。
