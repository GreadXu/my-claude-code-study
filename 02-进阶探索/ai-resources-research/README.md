## 模块概述

everything-claude-code 是一个全面收集 Claude Code 生态资源的仓库。本模块将浏览和研究这些资源，帮助你快速了解 Claude Code 生态的全貌，发现有用的工具和扩展。

## 常见问题

### Q: 所有资源都需要尝试吗？
**A**: 不需要。快速模式只需了解资源类型，完整模式建议选择 3-5 个感兴趣的资源深入尝试。

### Q: 如何选择有价值的资源？
**A**: 可以参考 GitHub Star 数、更新频率、以及与个人需求的匹配度。

### Q: 可以贡献自己的资源吗？
**A**: 可以。everything-claude-code 是开放仓库，欢迎提交 PR 添加新资源。
