## 模块概述

spec-kit 是 GitHub 推出的规范驱动开发工具包，通过预定义的规范模板和斜杠命令，帮助团队建立一致的开发流程。本模块将学习规范驱动开发理念和使用 spec-kit 进行项目开发。

## 常见问题

### Q: SDD 和敏捷开发冲突吗？
**A**: 不冲突。SDD 提供规范和模板，可以与敏捷开发结合使用，提高效率。

### Q: 必须在团队中使用吗？
**A**: 不是。个人项目也可以使用，有助于建立规范和提高效率。

### Q: 可以自定义模板吗？
**A**: 可以。spec-kit 支持自定义模板，可以根据团队需求调整。
