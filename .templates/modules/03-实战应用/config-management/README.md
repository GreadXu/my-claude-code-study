## 模块概述

cc-switch 是一个基于 Tauri 框架开发的 Claude Code 配置管理工具。本模块将学习配置管理的概念、Tauri 桌面应用架构，以及如何使用和贡献 cc-switch 项目。

## 前置知识

- [ ] 熟练使用 Claude Code
- [ ] 了解配置管理概念
- [ ] React 基础（可选，用于理解前端）
- [ ] Rust 基础（可选，用于理解后端）

## 常见问题

### Q: 必须会 Rust 才能学习吗？
**A**: 快速模式不需要。完整模式下，了解 Rust 会更有帮助，但不是必须的。

### Q: cc-switch 只能用于 Claude Code 吗？
**A**: 主要为 Claude Code 设计，但配置管理的思路可以应用到其他工具。

### Q: 可以为 cc-switch 贡献代码吗？
**A**: 可以。项目是开源的，欢迎提交 PR 或 Issue。
