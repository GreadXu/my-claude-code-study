## 常见问题

### Q: 必须按顺序学习吗？
**A**: practical-projects 是核心模块，建议优先完成。config-management 和 spec-driven-dev 是可选工具，可以根据需要学习。

### Q: practical-projects 模块做什么项目？
**A**: 聚焦于**AI 能力集成**，如配置多个 MCP 服务器、实现 AI 编排工具与 Claude Code 协作等。

### Q: 没有开发经验可以学吗？
**A**: 可以。本阶段重点是**配置和使用**而非开发。practical-projects 模块聚焦于配置多个 AI 工具协同工作。
