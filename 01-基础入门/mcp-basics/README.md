## 模块概述

MCP（Model Context Protocol）是 Anthropic 推出的开放协议，用于连接 AI 应用与外部数据源和工具。本模块将学习 MCP 的基础概念、工作原理，以及如何配置和使用基础的 MCP 服务器。

## 常见问题

### Q: MCP 和 API 有什么区别？
**A**: MCP 是专门为 AI 应用设计的协议，提供了标准化的工具发现和调用机制，而 API 是更通用的接口概念。

### Q: 需要编程基础吗？
**A**: 快速模式不需要。完整模式下，配置现有服务器也不需要编程，但开发自定义服务器需要 JavaScript/TypeScript 基础。

### Q: 配置 MCP 服务器安全吗？
**A**: MCP 服务器运行在本地，你完全控制数据流。只使用可信来源的服务器即可。
