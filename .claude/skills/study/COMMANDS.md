# 学习工作流程命令参考

## 概述

本学习计划使用 **自然语言触发** 的方式进行进度和模块管理。你只需要用自然语言描述你的需求，Claude Code 会根据 `CLAUDE.md` 中定义的工作流程自动执行相应操作。

---

## 学习管理命令

### 开始学习

**触发方式**（任意一种均可）：
- "开始学习 `<模块名>`"
- "启动 `<模块名>` 的学习"
- "开始学习模块 `<模块名>`"

**功能**：
1. 验证模块是否存在
2. 询问选择学习模式（快速/完整）
3. 显示更新预览
4. 更新 `checklist.md`、`notes.md`、`PROGRESS.md`
5. 添加学习日志

**示例**：
```
你: 开始学习 claude-code-core

Claude: [执行流程]
1. 模块已找到: claude-code-core [P0🔴 必学]
2. 请选择学习模式:
   - 1. 📚 快速模式（约 3 天）- 理论为主
   - 2. 🛠️ 完整模式（约 1 周）- 理论+实践
3. [显示预览并确认]
4. [应用所有更新]
```

---

### 更新进度

**触发方式**：
- "更新进度 `<模块名>`"
- "更新 `<模块名>` 的学习进度"
- "同步进度 `<模块名>`"

**功能**：
1. 读取 `checklist.md` 计算完成百分比
2. 显示进度变化预览
3. 更新 `PROGRESS.md` 中的进度
4. 添加进度日志

**注意**：你需要先手动编辑 `checklist.md` 勾选已完成的项目，然后运行此命令更新进度。

**示例**：
```
你: [手动编辑 checklist.md，勾选几个完成项]
你: 更新进度 claude-code-core

Claude: [执行流程]
1. 解析 checklist.md...
2. 快速模式: 4/12 项完成 (33%)
3. [显示预览并确认]
4. [应用更新]
```

---

### 完成学习

**触发方式**：
- "完成学习 `<模块名>`"
- "标记 `<模块名>` 为已完成"
- "完成模块 `<模块名>`"

**功能**：
1. 验证所有必选项是否完成
2. 如有未完成项，显示警告
3. 更新 `notes.md` 记录完成日期
4. 更新 `PROGRESS.md` 状态为"已完成"
5. 添加完成日志

**示例**：
```
你: 完成学习 claude-code-core

Claude: [执行流程]
1. 验证完成情况...
2. [如果有未完成项]
   ⚠️ 警告: 以下项目尚未完成:
   - [📚 快速模式] 完成 10 个核心命令练习
   是否仍要完成? (y/n):
3. [应用所有更新]
```

---

### 查看状态

**触发方式**：
- "查看学习状态"
- "显示进度概览"
- "学习进度怎么样了"

**可选指定模块**：
- "查看 `<模块名>` 的状态"
- "显示 `<模块名>` 的进度"

**功能**：
- 显示所有模块的进度概览
- 显示各优先级进度
- 显示每个模块的状态卡片

**示例**：

...

## 模块管理命令

### 新增模块

**触发方式**：
- "新增模块 `<模块名>`"
- "添加模块 `<模块名>`"
- "创建新模块 `<模块名>`"

**功能**：
1. 创建模块目录结构
2. 生成模板文件（README.md、checklist.md、notes.md）
3. 更新 PROGRESS.md 添加模块行
4. 更新 CLAUDE.md 添加模块映射

**示例**：
```
你: 新增模块 my-new-module

Claude: [执行流程]
1. 请输入模块信息:
   - 模块名: my-new-module
   - 所属阶段: 02-进阶探索
   - 优先级: P1 推荐
   - 预计时长: 1 周
2. [显示预览并确认]
3. [创建目录和文件]
4. [更新 PROGRESS.md 和 CLAUDE.md]
```

---

### 更新模块信息

**触发方式**：
- "更新模块信息 `<模块名>`"
- "修改模块 `<模块名>`"
- "编辑模块 `<模块名>`"

**功能**：
1. 读取模块当前信息
2. 询问要修改的字段
3. 显示变更预览
4. 更新相关文件

---

### 重置模块

**触发方式**：
- "重置模块 `<模块名>`"
- "重新开始 `<模块名>`"
- "清除 `<模块名>` 的进度"

**功能**：
清除模块的所有学习进度，恢复到未开始状态

**示例**：
```
你: 重置模块 claude-code-core

Claude: ⚠️ 警告: 此操作将清除模块的所有学习进度
       是否继续? (y/n):
```

---

### 删除模块

**触发方式**：
- "删除模块 `<模块名>`"
- "移除模块 `<模块名>`"

**功能**：
删除模块目录和所有相关文件，从系统中移除模块

**示例**：
```
你: 删除模块 old-module

Claude: ⚠️ 警告: 此操作将删除模块目录和所有文件
       请输入模块名确认删除: old-module
```

## 学习模式

### 快速模式 📚
- **时长**：约 30% 完整模式时间
- **内容**：理论学习为主
- **输出**：学习笔记总结

### 完整模式 🛠️
- **时长**：模块标注时长
- **内容**：理论 + 全部实践练习
- **输出**：学习笔记 + 练习代码 + 作品

## 典型使用流程

### 1. 开始新模块
```
你: 开始学习 claude-code-core
[选择快速模式]
[确认]
```

### 2. 完成学习项
```
你: [编辑 01-基础入门/claude-code-core/checklist.md]
    [勾选已完成的项目]
你: 更新进度 claude-code-core
```

### 3. 完成模块
```
你: 完成学习 claude-code-core
[如有未完成项，确认是否强制完成]
```

## 优先级说明

| 优先级 | 标签 | 说明 | 推荐模式 |
|--------|------|------|----------|
| P0 | 🔴 必学 | 核心基础，必须掌握 | 完整模式 |
| P1 | 🟡 推荐 | 重要内容，强烈推荐 | 完整模式 |
| P2 | 🟢 可选 | 根据兴趣选择 | 快速/完整 |
| P3 | 🔵 了解 | 理论浏览即可 | 快速模式 |

## 错误处理

### 模块未找到
```
你: 开始学习 unknown-module
Claude: ❌ 错误: 未找到模块 "unknown-module"
       可用模块: [列出所有模块]
```

### 模块已开始
```
你: 开始学习 claude-code-core
Claude: ⚠️ 该模块已经开始学习
       当前模式: 快速模式
       如需重新开始，请先手动重置 checklist.md
```
