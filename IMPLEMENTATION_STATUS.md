## 🔧 技术说明

### 为什么使用自然语言而不是 Skill？

1. **Skill 系统限制**
   - `.claude/skills/` 目录不会自动被识别
   - 需要使用官方 `/create-skill` 命令创建
   - 当前该命令不可用或未验证

2. **CLAUDE.md 优势**
   - Claude Code 会自动读取并遵循 CLAUDE.md 中的指令
   - 自然语言触发更灵活，无需记忆精确命令
   - 易于维护和更新

3. **实现方式**
   - 在 CLAUDE.md 中定义完整的工作流程
   - AI 根据用户输入的自然语言匹配对应模式
   - 执行相应的文件操作和状态更新

### 进度计算逻辑

```python
# 伪代码
def calculate_progress(checklist_content):
    # 检测学习模式
    if is_quick_mode_selected():
        items = get_items_under("📚 快速模式")
    else:
        items = get_items_under("🛠️ 完整模式")

    # 统计完成数
    total = len(items)
    completed = count_checked(items)

    # 计算百分比
    progress = (completed / total) * 100
    return progress
```
